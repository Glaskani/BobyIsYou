Attention pour l'ajout d'une carte il faut ajouter le fichier "TXT" dans code/src/common/ressources/maps.
Puis dans le fichier java "Levels" ajouter dans la liste à la ligne 146 le nom dans le tableau de char.

Dans un fichier de map, il faut impérativement la dimension à la première ligne.
Ensuite optionnellement un titre (title Coucou_Je_Suis_Un_Super_Informaticien).
Enfin toujours optionnel une ligne "config" (config 50) cela represnet en game mode déplacement limiter le nombre max de déplacement.
